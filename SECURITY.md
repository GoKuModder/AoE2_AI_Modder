# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Email: <SECURITY_CONTACT_EMAIL>
- Alternate contact: open a GitHub issue with minimal details and request a private follow up.

Please include:

- A description of the issue and impact.
- Steps to reproduce.
- Affected versions or commit hash, if known.

## Supported versions

This project is currently developed from `main`. Security fixes, if needed, are applied to the latest code.
